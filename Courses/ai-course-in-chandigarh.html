Artificial Intelligence(AI) Syllabus

  • Overview of AI
  • What is AI? Definitions, history, and types of AI.
  • Key Concepts: Machine Learning (ML), Deep Learning, NLP, Computer Vision
  • Applications or Uses of AI in real life

  • Introduction & Syntax: Learn Python’s fundamental syntax, variables, data types, operators, and control structures.
  • Functions & Data Structures: Understand functions, common data structures (lists, tuples, dictionaries, sets), file handling, and error handling.

  • Classes & Objects: Create and use classes, objects, and constructors to structure code.
  • Core OOP Concepts: Apply key OOP principles such as encapsulation, inheritance, polymorphism, and abstraction in real-world scenarios.

  • Basic coding exercises: data manipulation, visualization, and working with datasets

  • Setting up Jupyter notebooks for hands-on coding
  • Basic exercises in data loading, cleaning, and visualization

  • Introduction to Machine Learning
  • Supervised vs. Unsupervised Learning
  • Key algorithms: Linear Regression, Logistic Regression, Decision Trees
  • Model evaluation: Accuracy, Precision, Recall, F1-Score

  • Importance of data preprocessing (handling missing data, outliers)
  • Feature scaling (Normalization, Standardization)
  • Encoding categorical variables (One-hot encoding, Label encoding)

  • Basics of Neural Networks
  • What are neural networks? Components of a neural network (input layer, hidden layers, output layer)
  • Activation functions (e.g., ReLU, Sigmoid, Softmax)

  • Overview of Backpropagation and Gradient Descent
  • Loss functions (Mean Squared Error, Cross-Entropy)
  • Introduction to TensorFlow/Keras for building neural networks

  • Introduction to Convolutional Neural Networks (CNNs) for image recognition
  • Introduction to Recurrent Neural Networks (RNNs) for sequence data (e.g., text, time-series)

  • Gain practical experience by working on AI-driven projects, including machine learning models, data analysis, and neural networks, to solidify your learning.
